# project-work-granarolo-sostenibilita-2026
Project Work - Tecnologie Web (Tema 3: Sostenibilità)
Per il mio progetto ho scelto il caso Granarolo, focalizzandomi sulla sostenibilità nella filiera del latte. L'idea era creare un sito molto semplice che mettesse in ordine i punti principali senza troppi giri di parole.

-Cosa c'è nel progetto:
Il sito è diviso in sezioni standard (header, chi siamo, azioni e footer) e si concentra su 5 aree: benessere animale, nutrizione, pratiche agronomiche, impatto ambientale e il progetto biometano. Non volevo riempire le pagine di testo inutile, quindi ho cercato di sintetizzare il tutto per dare un quadro chiaro della situazione.
Il sito è online qui: https://limone03.github.io/project-work-granarolo-sostenibilita-2026/
-Scelte tecniche:
Per lo sviluppo ho usato HTML e CSS puro. Ho preferito evitare framework esterni per fare in modo che il codice rimanesse leggero. Per il layout ho usato CSS Grid, che mi è sembrato il modo più comodo per gestire le card in modo ordinato. Il sito è responsive, quindi si adatta anche se visualizzato da cellulare. Ho aggiunto un'ombra e un effetto hover sulle card per dare un minimo di interazione, ma cercando di non appesantire il design.
-Struttura dei file:
index.html: la pagina principale con l'HTML.
styles.css: il foglio di stile (ho cercato di tenerlo pulito).
/report: cartella che contiene il PDF con l'analisi completa che ho collegato al sito.
